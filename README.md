# 🤖 CodeAlpha AI Internship - Task 2: AI FAQ Chatbot

A high-performance, retrieval-based chatbot that utilizes custom Natural Language Processing (NLP) to resolve internship-related FAQs.

## 🚀 Live Demo
Test the chatbot interface here:  
👉 **[View Live Demo](https://mohannadghader.github.io/CodeAlpha_Chatbot_FAQ/)**

## ✨ Technical Highlights
- **Custom NLP Pipeline:** Implemented tokenization, normalization, and stop-word filtering using Vanilla JavaScript to identify user intent.
- **Weighted Similarity Algorithm:** Developed a retrieval logic that calculates the best match between user queries and the FAQ database.
- **Sentiment & Small Talk:** The system recognizes basic user moods and greetings for a more natural conversational flow.
- **Interactive UI/UX:** Features a modern glassmorphism design with dynamic "Suggestion Chips" for common inquiries.
- **Responsive Architecture:** Fully optimized for both desktop and mobile viewing.

## 🛠️ Technologies Used
- **Logic:** Vanilla JavaScript (ES6+)
- **Styling:** CSS3 (Glassmorphism & Flexbox)
- **Structure:** HTML5
- **Deployment:** GitHub Pages

## 📂 Project Files
- `index.html`: Chat container and interactive elements.
- `style.css`: Glassmorphism effects and layout styling.
- `script.js`: The NLP engine and chatbot response logic.
